Para que o site rode de forma correta é necessário momentaneamente baixar o Wampserver(http://www.wampserver.com/en/), nele criar dois bancos de dados MySQL.

No Primeiro é necessario que se chame "novobancodados", nele devera conter as seguintes informações:

nome
sobrenome
CPF
RG
CRM
pais
estado
cidade
email
senha
ID

Já o segundo Banco devera se chamar "site" ser criado os campos:
input_User
input_Pass
Id

Esses metodos são validos para que as informações sejam salvas, acessadass e validadas a partir do banco!

As informações de portas de acesso contidas nos arquivos do site, deveram ser validadas de acordo com suas configurações do Wamp Server.
